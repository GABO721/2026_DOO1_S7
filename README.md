## Descripción general del sistema

Este proyecto trata de una aplicación de consola desarrollada en java para la agencia Llanquihue Tour. Su objetivo principal es mostrar en consola los datos de instancia de cada clase derivada de la superclase "Servicio Turistico". Se aplicaron conceptos de herencia y polimorfismo.

---

## Estructura general del proyecto

```plaintext
📁 src/
├── data/     # Clase de lógica encargada de instanciar
├── model/    # Jerarquía de clases
├── ui/       # Clase principal con el método main
````

---


## Instrucciones ejecutar el proyecto

1. Descarga el proyecto.

2. Abre un editor de codigo.

3. Ejecuta el archivo `Main.java` ubicado en `src/ui/Main.java`.
